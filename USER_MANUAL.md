# 健身计划管理应用 - 用户手册

欢迎使用你的专属健身计划管理应用！本应用旨在帮助你轻松制定健身目标、跟踪执行情况，并通过微信保持提醒和定期报告。

## 1. 核心功能说明

### **1.1 健身计划 (Plans)**
- **每周/每月计划**: 设定固定日期的锻炼项目。
- **提醒设置**: 为每个计划设置具体的提醒时间（如 08:30）。
- **确认链接**: 微信推送的消息中包含一个专属确认链接，点击后即可标记为已完成。

### **1.2 临时活动 (Activities)**
- **一次性安排**: 突然想加练？在这里添加。
- **单独提醒**: 临时活动也有专属提醒。

### **1.3 待办事宜 (To-dos)**
- **健身周边**: 比如“购买筋膜枪”、“准备运动餐”等日常小事。

### **1.4 统计分析 (Analysis)**
- **可视化总结**: 查看本周/本月的总完成数。
- **数据汇总**: 所有数据自动存储在 D1 数据库中。

## 2. 微信提醒与推送

### **2.1 推送消息样例**
> **健身提醒: 腿部训练**
>
> 深蹲、硬拉等核心动作，记得热身！
>
> 确认完成: [https://your-app.pages.dev/confirm-plan/123]

### **2.2 自动报告**
- **周报**: 每周一早上 09:00，微信将推送上周的健身总结。
- **月报**: 每月 1 号早上 09:00，微信将推送上月的成就单。

## 3. 使用技巧

- **精准提醒**: 请确保 Cloudflare Worker 的定时触发器（Cron Trigger）配置正确。
- **个性化通知**: 如果你想更改推送内容，可以联系开发者修改 `worker.ts`。
- **查看分析**: 所有的统计结果都可以在应用的“分析”选项卡中实时查看。
